Pour une utilisation et une manipulation optimale des dossiers / fichiers / données du projet voici quelques conseils et recommandations :

La Documentation :

	Toute la Documentation se trouve dans le répertoire "Documentation".
	A l'interieur de ce dernier se trouve 3 répertoires :
		- Contexte (Où se trouve le document contenant les différentes explications / justifications,
		 diagrammes disponible en docs ET en PDF).

		- Diagram (Où se trouve trois répertoires avec les diagrammes de classes, d'utilisations. Dans
		 chacun des répertoires se trouvent les fichiers StarUML mais également les diagrammes en format PNG.
		Vous pouvez donc consulter les Diagrammes ici si ceux sur le Contexte ne sont pas bien lisible.

		- Sketch (Où se trouve les différentes sketch de l'application).

Le Code :

	Rappel : La configuration de votre IDE est indépendante au développement du projet, il devrait donc pouvoir se lancer sous
	n'importe quel IDE pour Java.

	L'application a été développée sous JAVA 11, donc merci de bien vouloir lancer l'application avec le SDK adéquat.

	Le répertoire "out" n'a pas été mit sur le Git donc avant le démarrage de l'application, merci de créer un répertoire nommé "out"
	et de le configurer comme répetoire de sortie du compilateur (Project Compiler Output).

	Le répertoire "res" contient les pages FXML necessaire à l'application, les pages CSS ainsi que les images necessaires aux Skins de 
	personnages, l'application ne peut pas être lancée en mode "Console".
